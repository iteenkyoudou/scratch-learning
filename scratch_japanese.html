📦 ひつようなブロックをよういしてください

したのブロックのなかから、プログラムにひつようなブロックをえらんでください。

@greenFlag が押されたとき :: events
[こんにちは！] と言う :: looks
(50) 歩動かす :: motion
[さようなら！] と言う :: looks
右に (90) 度回す :: motion
💡 ヒント: みどりのはたをクリックしたときからはじまるプログラムをつくりましょう！

🔧 このようにくみたててください

ただしいじゅんばんでブロックをくみたてると、つぎのようになります：

せいかいれい:

@greenFlag が押されたとき :: events
[こんにちは！] と言う :: looks
(50) 歩動かす :: motion
右に (90) 度回す :: motion
[さようなら！] と言う :: looks
✅ このじゅんばんでブロックをくみたてると、プログラムがただしくうごきます！

▶️ プログラムのじっこう

プログラムはうえからじゅんばんにじっこうされます。どのようなじゅんばんでうごくかみてみましょう！

じっこうじゅんじょ:

1
@greenFlag が押されたとき :: events
2
[こんにちは！] と言う :: looks
3
(50) 歩動かす :: motion
4
右に (90) 度回す :: motion
5
[さようなら！] と言う :: looks
📚 プログラムはかならずうえからした、じゅんばんどおりにじっこうされます。このじゅんじょがたいせつです！
