📦 必要なブロックを用意してください

下のブロックの中から、プログラムに必要なブロックを選んでください。

when flag clicked :: events
はたがおされたとき
say [Hello!] :: looks
こんにちは！という
move (50) steps :: motion
50ぽうごかす
say [Goodbye!] :: looks
さようなら！という
turn cw (90) degrees :: motion
みぎに90どまわす
💡 ヒント: 緑の旗をクリックしたときから始まるプログラムを作りましょう！

🔧 このように組み立ててください

正しい順番でブロックを組み立てると、次のようになります：

正解例:

when flag clicked :: events
say [Hello!] :: looks
move (50) steps :: motion
turn cw (90) degrees :: motion
say [Goodbye!] :: looks
1. はたがおされたとき
2. こんにちは！という
3. 50ぽうごかす
4. みぎに90どまわす
5. さようなら！という
✅ この順番でブロックを組み立てると、プログラムが正しく動きます！

▶️ プログラムの実行

プログラムは上から順番に実行されます。どのような順番で動くか見てみましょう！

実行順序:

1
when flag clicked :: events
はたがおされたとき
2
say [Hello!] :: looks
こんにちは！という
3
move (50) steps :: motion
50ぽうごかす
4
turn cw (90) degrees :: motion
みぎに90どまわす
5
say [Goodbye!] :: looks
さようなら！という
📚 プログラムは必ず上から下へ、順番通りに実行されます。この順序が大切です！
